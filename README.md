## 프로젝트 소개

밥짝은 혼밥이 외롭거나 자연스러운 교류가 필요한 대학생들이
식사 스타일·시간·관심사 기반으로 가볍게 ‘밥약’을 만들 수 있는 매칭 서비스입니다.

- 즉석 밥약 / 선후배 밥약 / 같은 수업 밥약

- 가벼운 밥 한 끼로 확장되는 대학 내 네트워킹

- 부담 없는 UI, 단순한 UX 흐름, 빠른 매칭 경험

> “같은 학교, 비슷한 상황의 학생끼리 자연스럽게 이어지는 식사 연결 서비스”

**본 리포지토리는 밥짝 서비스의 백엔드 리포지토리입니다.
#### 프론트엔드 리포지토리: https://github.com/25Eiiii/Eiii-f

## 핵심 기능
### 로그인 및 선호 정보 입력
<img width="375" height="166" alt="image" src="https://github.com/user-attachments/assets/eb5d985c-87c8-4d1a-9e08-5ea2d7f945cf" />
<br>
- 사용자 맞춤 매칭과 추천을 위해 선호식, 학번, 관심사 등 최소한의 정보를 단계별로 입력받도록 설계

### 매칭 탐색 및 신청
<img width="299" height="161" alt="image" src="https://github.com/user-attachments/assets/e43eb073-3cef-4de7-9cdd-dc8de75cae77" />
<br>
- '즉석 밥약', '선후배 밥약', '같은 수업 밥약' 등 매칭 목적별 탐색 경로를 분리하여 목적성 있는 사용 유도


### 대화 및 커뮤니티
<img width="299" height="161" alt="image" src="https://github.com/user-attachments/assets/e43eb073-3cef-4de7-9cdd-dc8de75cae77" />
<br>
- 매칭 이후 사용자 간 교류 유지를 위해 ‘쪽지 요청 → 수락 → 대화’ 흐름을 설계
<br>
- 커뮤니티 기능은 단순 밥약 외의 교류 유도를 위한 공간으로 활용
<br>
- 좋아요/스크랩/댓글 기능으로 참여 유도와 관심 정보 저장 기능 제공
